STL Map 教學

在C++基本函式庫中實作紅黑樹(Red Black Tree)和雜湊(Hash)的資料結構分別是
map 和 unordered_map 兩個資料結構在使用上基本上沒有什麼區別，不同的部分只有map 的紅勒樹是可以用iterator尋訪，以及搜尋時間是O(log N) 而unordered_map則不能尋訪但其搜尋時間是
O(1)。
