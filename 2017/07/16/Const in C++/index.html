3.function

通常會用到這個都是在class的member function上，不過要記得static member function後頭不能加上const喔!!

[1] 和 [2] 都是指回傳的數值會是常數。[3] 則是整個member function中，都不可以更動到物件成員。所以如果將const放在[3]的位置上的化這個範例就會產生complier error
