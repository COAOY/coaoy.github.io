GNU GCC on Windows

2017-07-19

檔案安裝

工欲善其事，必先利其器。在安裝GCC之前我們要先把好用的文字編輯器安裝好。我個人推薦的是visual studio code 和 Sublime text 兩個個有優缺，喜歡那個就裝那一個吧!!

visual studio code
Sublime Text 3

接下來要下載GNU GCC了，下面我提供懶人載點不過最好還是自己上網找最新的比較好。

key word : mingw-get-inst
GNU GCC

make 一樣提供懶人載點，雖然很久沒更新了不過還是找找看比較好。

key word : make for windows
make

更改環境變數

更改環境變數的用意在於讓終端機知道要去哪裡找執行指令的文件。

步驟:windows 的系統 >> 進階系統設定 >> 進階 >> 環境變數 >> 系統變數裡的Path >> 編輯。

在最後加上分號然後貼上你安裝的MinGW的bin資料夾的完整路徑，預設是C:\MinGW\bin。加上分號後再貼上GNU32的bin資料夾的完整路徑，預設是C:\Program Files (x86)\GnuWin32\bin。

之後打開你的終端機(搜尋 >> CMD)打上gcc -v 看看有沒有出現版本資訊



終端機捷徑製作

創立一個新的文字檔，內容打cmd >> 將主檔名改成Terminal附檔名改成.bat >> 雙擊兩下打開檔案。
